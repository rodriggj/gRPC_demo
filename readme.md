# gRPC

## Steps
1. Initiate a `NodeJS` project

```javascript 
npm init -y 
```

2. In order to utilize protocol buffers you need to initiate a file that will define the `Schema` of the API that you are going to utilize in your application. 

```javascript 
code todo.proto
```
